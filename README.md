# 中华护理学会 自动刷课

自动完成中华护理学会继续教育平台的课程学习。

## 功能

- 🎬 倍速播放（1x ~ 16x 可调），加速剩余时间实时计算
- 🔇 自动静音
- 📝 自动答题（多题连答）
- ⭐ 五星评分 / 满意度自动选择
- 🔗 自动切下一节 → 全部播完 → 打分 → 跳到下一门课
- 📦 面板可拖动，不遮挡页面内容

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开 [中华护理协会刷客脚本.user.js](中华护理协会刷客脚本.user.js)
3. 点击 Tampermonkey 弹出的安装按钮

## 支持平台

- `study.zhhlxh.org.cn` — 课程学习
- `course.zhhlxh.org.cn` — 课程评分
